# IntegrityWeb Platform - Complete Page Guide

## Overview
IntegrityWeb is a modern, developer-focused platform for building verifiable AI applications. The platform features a sophisticated landing page, comprehensive resources, and an administrative dashboard.

## Navigation Structure

### Public Pages (Main Navigation)
- **Docs** - Documentation and API reference
- **Tools** - Collection of development tools
- **Explore dApps** - Directory of applications
- **Build** - Building guides and frameworks

### Dashboard Pages (When Logged In)
- **Dashboard** - Metrics and network monitoring
- **Proofs** - Validity proofs explorer
- **Ledger** - Record ledger and history
- **Network** - Network resilience map
- **Agents** - Agent identity management
- **Privacy** - Zero-knowledge shield settings

## Page Descriptions

### 1. Landing Page (`/`)
**Purpose:** Showcase IntegrityWeb's value and convert visitors
**Features:**
- Hero section with call-to-action
- Three Axioms section (Code is Math, Math is Reality, Reality is Verifiable)
- "What You Can Build" section featuring 6 use cases
- Getting Started section with three pathways
- Social proof showing support from developers, startups, AI agents, and communities
- CTA section encouraging community participation

### 2. Dashboard (`/dashboard`)

**Features:**
- Integrity Metrics display (Proofs Generated, Integrity Score, Network Hash Rate, Active Nodes)

## Design System

### Colors
- Primary: Cyan/Teal (#75 0.15 185° in OKLch)
- Secondary: Dark backgrounds
- Accent: Lighter cyan
- Destructive: Red (#55 0.22 25° in OKLch)

### Typography
- Font: Geist (sans-serif)
- Mono: Geist Mono (for code)
- Sizes: Responsive from mobile to desktop

### Layout
- Flexbox-based layouts
- Container max-width: 5xl (64rem)
- Consistent padding and gaps
- Mobile-first responsive design
