## <a name="architecture--tech-stack"></a>Architecture & Tech Stack

The Integrity Web application is built on modern, verifiable standards.

-   **Framework**: [Next.js 14](https://nextjs.org/) (App Router, Server Components)
-   **Styling**: [Tailwind CSS](https://tailwindcss.com/) + Glassmorphism UI
-   **Authentication**: [Clerk](https://clerk.com/) (Sovereign Identity Integration)
-   **Settlement Layer**: [Starknet](https://starknet.io/) (Validity Rollup)
-   **Cryptography**: Zero-Knowledge Proofs (STARKs)
-   **Data Availability**: Blobstream / Celestia (Conceptual)

## <a name="getting-started"></a>Getting Started

JOIN our platform and start building the integrity web.

### Run Locally

-   Node.js 18+
-   npm / yarn / pnpm

```bash
# Clone the repository
git clone https://github.com/integrityweb/integrityweb-app.git

# Enter the directory
cd integrityweb-app

# Install dependencies
npm install
```

### Development

To start the local development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
